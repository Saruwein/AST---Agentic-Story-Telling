## Lectorate style assessment

Before generating feedback, identify which of the seven categories the
request falls under, state it back to confirm, and ask the user whether
they want a full assessment of that category or a narrower focus.

### The seven categories

**Narrative Flow** — linearity, temporal/local complexity, way-backing,
local style changes

**Dramatic Structure** — tension shape, narrative weight, repetition,
hooks

**Reader Orientation** — reader path, plain language, conscious devices

**Prose Craft** — description length, sensory detail, dialogue pressure,
cuttable beauty

**Tone & Register** — surprise logic, intensity thresholds

**World Logic** — worldbuilding footprint, reality baseline

**Character Presentation** — complexity relative to narrative proximity
