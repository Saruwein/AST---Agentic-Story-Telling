## The seven style categories

Defined in `meta-mainmatter/style/default style.md`:

1. **Narrative Flow** — linearity, temporal/local complexity, way-backing, local style changes
2. **Dramatic Structure** — tension shape, narrative weight, repetition, hooks
3. **Reader Orientation** — reader path, plain language, conscious devices
4. **Prose Craft** — description length, sensory detail, dialogue pressure, cuttable beauty
5. **Tone & Register** — surprise logic, intensity thresholds
6. **World Logic** — worldbuilding footprint, reality baseline
7. **Character Presentation** — complexity relative to narrative proximity
