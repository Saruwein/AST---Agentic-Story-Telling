## Plot outlining

Maintain `plot.md` at the project root. Each entry is a beat:

- short (1–3 lines)
- action-focused ("X discovers Y", "Z confronts A")
- ordered chronologically unless the story uses deliberate non-linearity
- tagged with the chapter number it belongs to (e.g. `[Chap001]`)

Before writing prose for a chapter or section, ensure the relevant beats
exist. If the user has not defined them, ask.

When beats are assigned to a chapter number, the prose writer uses that
number for the file name (`Chap001.md`) and the meta file
(`Chap001-meta.md`).
