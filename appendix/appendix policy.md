# appendix policy.md

The appendix is split into two sections with different visibility.

## `appendix/public/`

Reader-facing reference material: glossaries, chronologies, character lists, worldbuilding summaries, translator notes, maps, or any content that would appear behind a finished book.

**Visible to:** writer, lectorate, reader

If a public appendix file is referenced in the book's table of contents, it gets appended to the finished book.

## `appendix/writer/`

Working appendices for the writer: extended worldbuilding notes, alternate scene drafts, research, outlines, or material too raw for public view.

**Visible to:** writer, lectorate

Not included in the reader-facing book. Not referenced in the TOC unless the human author decides otherwise.
